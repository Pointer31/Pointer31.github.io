Another fun server mod of Teeworlds 0.7! It adds new gametypes including instagib, as well as map teleports and bots!

You may find the v0.9.0 versions on the github. Right now, there is also a 0.7-based version being worked on!

Features

Original Teeworlds gametypes!

DM, TDM, CTF, LMS, LTS!


And instagib variants!

        
Instant kills with grenades or lasers!


Two brand new gametypes!

Hide and Seek


Seekers will try to kill the Hiders, Hiders need to run!


Hold the Flag


Hold the flag to gain points over time!


Server-side bots!

bots pew pew, with multiple difficulties and 2 types! They can be added and removed automatically while there are few human players!


Map teleports!

:O kinda not intuitive to use though. They work differently than ddrace teleports, being more smooth and maybe more suited for PvP maps! (and more confusing to place in a map as mentioned)


For server hosters

what type of instagib you want can be set with sv_instagib. 0 for no instagib, 1 for laser instagib and 2 for grenade instagib. It will take effect on a reload. Note that hide and seek is not affected by this.

To set the custom gametypes use sv_gametype "htf" for hold the flag or "hns"/"hidnsek" for hide and seek.

To automatically add bots, set sv_bots_minimum_players. The server will then add as many bots as are required to have that many people playing, at least. When somebody joins or leaves, it will change the amount of bots appropriately.
