The duckclient! Based on ddnet 19.5 and the infclass client for ddnet/teeworlds, made by Pointer! It aims to add some fancy particles and add back some old UI visuals, while having features of the infclass client too! It also has a few cherry-picked features from newer or other clients. Among some other things too, like adding more ducks!

You may find the v4.1 versions below!

Download (Linux)
Download (Windows)
Download (macOS)

Features include:
  • Several toggles for old versions of UI!
  • Extra particles!
  • Self kill protection!
  • Features of the infclass client!
  • Ducks!
Most of the added config options can be found in the new DuckC settings tab.
