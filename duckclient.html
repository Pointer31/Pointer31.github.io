The duckclient! Based on ddnet 19.3 and the infclass client for ddnet/teeworlds, made by Pointer! It aims to add some fancy particles and add back some old UI visuals, while having features of the infclass client too! Among some other things too, like adding more ducks!

You may find the (outdated!) v1 versions below. There is no MacOS build, but you should be able to build it yourself.

Download (Linux)
Download (Windows)

Features include:
  • Several toggles for old versions of UI!
  • Extra particles!
  • Self kill protection!
  • Features of the infclass client!
  • Ducks!
