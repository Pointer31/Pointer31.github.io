Duck Client

The duck client! Based on the ddnet or infclass client for ddnet/teeworlds! Made by Pointer! Has exciting features!
There are no releases yet, but you can find the github repo's below.

Github (infclass-based)
Github (ddnet-based)


Features include:
  • Several toggles for old versions of UI!
  • Extra particles!
  • Ducks!
The infclass client based version (obviously) also includes various features useful for while playing the infclass gamemode.
