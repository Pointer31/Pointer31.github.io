The duckclient! Based on the infclass client for ddnet/teeworlds, made by Pointer! It aims to add some fancy particles and add back some old UI visuals, while having features of the infclass client too! Among some other things too, like adding more ducks!

You may find the v1 versions below. There is no MacOS build, but you should be able to build it yourself.

Download (Linux)
Download (Windows)

Features include:
  • Several toggles for old versions of UI!
  • Extra particles!
  • Self kill protection!
  • Features of the infclass client!
  • Ducks!
List of config options added by the duckclient:

cl_extra_particles
cl_old_door_laser
cl_old_freeze_laser
cl_old_client_console
cl_blood_particles
cl_position_community_filter
cl_confirm_kill_time
cl_duck_filter
